Simulación Monte Carlo aplicada al Transporte

Este proyecto muestra cómo utilizar simulación Monte Carlo para analizar costos y retrasos en operaciones de transporte, incorporando incertidumbre realista y enfocándose en toma de decisiones operativas bajo riesgo.
El objetivo principal es demostrar por qué las medias no son suficientes para presentar resultados operativos y cómo los percentiles permiten una planeación más robusta.

Objetivos del proyecto

* Simular la variabilidad en costos de transporte
* Modelar retrasos operativos mediante distribuciones probabilísticas
* Analizar resultados usando percentiles (P50, P75, P90, P95)
* Visualizar riesgos operativos y financieros
